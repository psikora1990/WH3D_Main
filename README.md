# WH3D_Main

Prototype implementation of the warehouse **core game loop**:

1. Create warehouse
2. Hire employees
3. Buy equipment
4. Take delivery
5. Storage
6. Pickup
7. Shipment
8. Profit
9. Expansion
10. Loop

## Run

```bash
python3 core_game_loop.py
```

This runs a short multi-day simulation and prints the loop events for each day.
